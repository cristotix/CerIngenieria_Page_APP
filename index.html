CerIngenieriaApp - Descarga Interna

Bienvenido/a al portal de descarga de nuestra aplicación interna. Esta versión está destinada **únicamente al personal de CER Ingeniería**.

Descargar Aplicación (Android .APK)

Instrucciones Importantes para la Instalación:

  1. Haz clic en el botón de descarga para obtener el archivo **.APK**.
  2. Si tu dispositivo lo solicita, debes **permitir la instalación de aplicaciones de orígenes desconocidos** (esto es normal fuera de Play Store).
  3. Una vez descargado, abre el archivo para iniciar la instalación.
  4. **AVISO:** Si ya tienes una versión instalada, es posible que debas desinstalarla primero.

Para soporte, contacta al equipo de TI.
